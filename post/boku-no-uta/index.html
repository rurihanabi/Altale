整合一下自己开播之后准备唱的歌

2022-02-12 · 14 min read

嗯，按照，嗯...想要唱的程度来分一下类吧。毕竟这样的话也更有逻辑，什么歌适合自己唱，什么歌自己能唱好，什么歌是单纯想而不适合之类的。

（10.22更新 果然在记录之前 还是应该尽可能把网易云的歌甚至B站的歌都转移到本地，下架打压太厉害了 可能按照我的速度根本跟不上了）

想唱！：
なんでもないや／RADWIMPS《没什么大不了》，听的版本是Eli的，这首歌速度不快，我应该能唱好。
银の龙の背に乗って《骑在银龙的背上》，听的版本是美雪阿姨的原版，这首歌以前唱过，不坏。
僕が死のうと思ったのは《我也曾想过一了百了》，听的是中岛美嘉阿姨的原版，以前唱过的歌，需要练习，未来能唱。
童話迷宮《童话迷宫》，听的是艾莉柯妮法的翻唱，不太容易唱吧，但是主要是因为想要整个像素小人，以后用得上。整体应该也行。
地球をあげる《把世界送给你》，也是Eli唱的版本，Eli的歌基本选的都是我自己感觉能做到的类型。一会儿至少还得有一首蝶。
瑠璃色の地球《琉璃色的地球》，艾莉柯妮法Ver.的，变貌大妃好玩法，未来一定要试试。歌很慢，我能唱好。
U＆I《U＆I》，听的是童田明治的版本。歌曲节奏挺快，但是在码字的时候听了一遍，看歌词能直接唱出来。应该没问题。
Dance Robot Dance《Dance Robot Dance》，听的版本是猫宫日向和隐神木荫唱的。这首歌我想找个朋友一起唱的。可惜唱不了了，估计要找别人了。
たばこ《烟草》，Eli唱的，不需要解释，我可以的。
シャングリラ《香格里拉》，J-POP的，虽然听的是YuNi的版本。感觉自己唱出来非常有意思的一首歌。留一下。
レッツゴー！陰陽師《新 豪血寺一族 -煩悩解放-》。虽然纯粹的是整活儿，但是我觉得这个真能唱。找人唱或者自己玩变貌大妃唱。
君に会うために《只为与你相见》。非常古早的歌了，当时还计划过半首唱中文，半首唱日文的。不过现在虽然退坑了舰R，但是还是想唱歌啊。想把自己的歌声...
我有一个装满星星的口袋《我有一个装满星星的口袋》，虽然是毕业生晚会的时候...自己唱虽然没有合唱的感觉了，但是估计也会别有一番风味。
W《W》，松冈充原版。假面骑士W的主题曲，E哥真帅，歌词也好。我的心中住着软弱，越长大一分，就更大一分。
鹿 be free《鹿 be free》，熊出没奇幻空间的插曲吧？第一次听的时候就很喜欢，国语歌唱唱这种也还行。
魂のルフラン《魂之轮回》，EVA，但听的不是原版。魂之轮回肯定要唱的。
不要过来！《不要过来！》，疯猴pme做的鬼畜。怎么说呢...哈哈哈哈哈，我居然想唱这个。很久没看过疯猴的鬼畜了，可以说是我看连贯鬼畜以来最喜欢的一个up了，权且当是致敬吧。
祭りのあと《祭典之后》，梁濑奈奈的歌，一首普通的，慢慢的歌。我觉得挺好听的，唱歌不久的话，从慢歌入手比较好呢。
远い约束《遥远的约定》，和上一首一个作者，也是慢慢的歌。啊，我真是个软弱的胆小鬼啊… 一厢情愿地把所有的好给了别人，假装自己什么都不缺，其实空荡荡的内心是多么渴望能被填满。 一个人的时候，多么希望能遇见一个人，和我一起被时间遗忘。 一个人的时候，假装自己习惯一个人。 当有人尝试靠近时，我却转身躲了起来。
瞬きもせず《一刹那》，中岛美雪阿姨的歌。你应该歌颂你自己，你是这么伟大；你应该喜欢你自己，因为我是这么喜欢你。
エイリアンエイリアン《Alien Alien》，偶像大师的歌吧？老实说这个也应该放到第二梯队的，但是就莫名的感觉自己能唱好这个。也放这儿。
Rave in the Grave《坟头蹦迪》，舞力全开的曲目。老实说比起唱这个，我应该是选择。。。跳一次这个。。。如果真的能凑到四个人的话就跳一次完整的吧。。。
ひかりもの《发光体》，あいみょん的，怎么这个都网易云收费了。前奏还没听出来什么，高潮的时候马上就想起来这是啥歌了。是一首自己感觉自己能唱好的歌。但实际怎么样以后再说吧。重点是要享受唱歌，而不是别的。就像我写字一样。
無口な明日を嫌わないで《请不要讨厌你少言寡语的明天》，山猿的。这个我行！感觉真行！
糸《糸》，原唱中岛美雪阿姨，但是这里还是选择用自己更喜欢的，超次元游戏海王星的ED3。这歌好听。老实说有一定的理由影响到了我出道的一些事情。这歌要唱的。
妄想Katharsis《妄想救赎》，新次元V2的真结局ED吧。黄油歌会唱过。唱过的歌自然感觉还能再唱。
Love Forever《Love Forever》，清水翔太和加藤米莉亚的歌。这个可以找人一起唱。很好听的一首歌。
葉桜《叶樱》，北乃きい的一首歌。好听的这个，虽然对自己来说有点挑战，但是想尝试一下。戻れないけどいい、戻らないけどいい。这句真不错。
永久の恋《永久之恋》，游魂KD第12话的ED。怎么说呢，泉户一家的爱情可以说是恋爱类的启蒙作品之一了吧。自己喜欢上兽耳萌娘也有一方面是这个原因。永久之恋到永久之樱，人与太转伊的千年之约最终交给了下一代。好歌，要唱的。“我是为了和裕理结合而来的”，“我是为了和真白结合而来的”，这个真好啊。下一个角色设计的时候多整点太转伊的特征吧）
Ring of Fortune《塑料内存条》OP，这个也是我童年的白月光了。但是真的越长大越看不得这张带刀的片子了。艾拉可爱。歌要唱的。祝你有一天能和你重要的人重逢。
朝焼けのスターマイン《塑料内存条》ED，和上一首一样，不解释。
過去と未来のメモリヘ《致过去与未来的记忆》，晴菜花綾崎菜乃花ED曲。在晴菜花OP和这个之间想了好久。。。咋说呢，歌是好歌，但是晴菜花吧。。。真不是给人玩的。
蒼い蝶《蓝色蝴蝶》，Eli的歌。我向她致敬。我已经不会再因为你离去而哭泣，但仍然想再看见你的背影，再听听你的声音。我会成为优秀的虚拟主播吧，像你一样。
far distance《遥远的距离》，晴菜花榊野木乃实ED曲。刚说完晴菜花这个就来了。操你妈，晴菜花我操你妈。别虐木乃实了行吗。いくつも重ねた未来へ掲げた、想いはあなたつながるためのdistance。你每句歌词都是踩在玩家心上啊，操你妈
僕らの手には何もないけど、《尽管我们的手中空无一物》，网易云居然没把这首歌变成收费的。感动。这首歌的MV真的是再冷漠的人也会动容吧。《镜中的笑容》这个电影以后也看看吧。
ふたつの影《二人一影》，这歌肯定要扔第二梯队的。但是想了想，呃，也不是不能挑战一下？丛雨线单线战士jpg
猛独が襲う《孤独猛袭》，原唱初音未来。但是听了几个版本都不是我想要的效果。未来试试我能不能唱好。
こわれかけのオルゴール《快要坏掉的八音盒》，小小的一个同人作品吧，我记得最早是。POP的画风让人印象深刻，2022年了还是那样。这是一作挺感人的...嗯。讲述了一个，呃，有关于救赎的故事吧。这么说好了。
涙のない世界《没有眼泪的世界》，AAA的歌我其实选了很多收，但是真的说实话，这些里我就感觉这个我能唱吧。可以挑战一下。
新木場発、銀河鉄道《新木場発、銀河鉄道》，《SB69》里听到的曲子。初见我就喜欢这个。声线舒服。
歌に形はないけれど《即使歌声无形》，听的是花姐的版本。老歌，想唱。
鳥の詩《鸟之诗》，某种意义上的国歌吧...这歌慢慢的，感觉自己能唱。
果てしない世界《没有尽头的世界》。嗯，DNF动画片的片尾曲，小时候就挺喜欢的，YMCK也是我喜欢的组合。这个未来可以试试，童年回忆jpg
小さな恋のうた《小小恋歌》，原唱新垣结衣，听过许多人的翻唱。这首歌基本可以说是谁唱都很好听了。这歌好，放在这儿。

春よ、来い《春天啊，到来吧》，松任谷由実原唱，但这个视频虽然写的是，实际上不是。是一首日语的老歌吧。感觉自己的声线唱这个还挺合适的。突然好喜欢杏花啊。
雪の華《雪之花》，原唱中島美嘉，但链接放的是虚拟猩猩兄妹的合唱。日语的抒情老歌我觉得我能唱好多。声线比较合适，恰好自己也喜欢。
うまぴょい伝説《俺の愛馬が！》。这首歌放在这里只有一个原因，许早苗我给你一拳。
Fairytaleは終わらない《永不完结的Fairytale》，Apprism专辑里的第九首，算是全专辑最喜欢的一首了。做好了搬走之后觉得自己还是唱不起来啊...草，但还是挑战一下吧，伴奏都怼我脸上了。

Daisuke《Daisuke》，IIDX的歌，daisuke~因为是真的很想做出那个动作啊（快把浴霸关上！
Imaginary affair《Imaginary affair》，《从此方至彼方》的OP，这个好听。老实说挺多gal曲都好听，但实在是没法唱出来。

想唱但是唱不了的类型？或者没有那么大欲望的歌：
惑星ループ《行星环》，这个...一方面是节奏有那么一点快，另一方面是声调应该不行，不过也不是不能尝试。
風になる《幻化成风》，听的是meaqua的版本。倒不是因为木口怎样的原因，就是感觉自己的心态不适合唱这个了。
ぼくらの"最☆強"おもちゃ箱ワンダーランド♪《我们的“最☆棒”玩具箱童话乐园♪》，nnhr的，在na组的时候就很喜欢的一首歌了。但是各种意义上，这首歌都唱不了吧（笑——
莎图温的裙摆《莎图温的裙摆》，我的声线不合适，变貌大妃也不合适，放在这儿。
狂おしいほど僕には美しい《狂谬即美丽》，majiko唱的。我觉得我真声唱不了这个，或者可能会崩——当然，恰恰相反也有可能。尽管说实在的，我觉得原版唱的不够狂乱。
卖鱼歌《卖鱼歌》，林北牛逼！整活向，也不是不行，但是肯定不是优先考虑的吧。我的粤语也没那么纯熟。
Emerald Sword《翡翠之剑》，我一直以为这是某个游戏的曲子呢。因为太带感了，加到清单里吧。现在唱英文歌和日文歌对我来说也差不多了。自己掌握的越来越多，也越来越有这种能力了。但是真说唱出这种感觉可太难了。
ハルジオン《春紫苑》，Yoasobi的。歌好听，Yoasobi的歌好多都很好听，但是基本都是我不太能唱的类型，即使考虑也是得考虑Acappella的唱法。
月燈籠《月灯笼》，nnhr的歌，不同于玩具箱，BPM191，整体难度都不小啊。歌本身很好，我给耳机试音的时候必用的一首曲子。但是真说唱好做出来，显然自己的技术力还远远不够。
たぶん《大概》，Yoasobi的歌。其实这首应该努努力可以唱。但是不作为第一梯队吧。
僕らの世代!《我们的世代》，节奏天国的歌。节奏天国是好游戏，希望大家都能玩玩。特别是对我这种晕一切3D的人来说，节奏感类的游戏可真是妙。啥时候节奏天国登录Switch了，我就去整个Switch去。
神様の渦《神明的漩涡》，Yunomi的。指定唱不了的类型。但是歌很喜欢，如果真的有足够的技巧了...呃，也不是不能考虑。
Climax Jump《假面骑士电王》主题曲。难度太高了！这个单人唱绝对唱不出来那种感觉吧！
キセキ《奇迹》，我觉得グリーンボーイズ版的更对自己胃口一点。歌是好的，我是烂的，躺平orzz
始まりの唄《始之歌》，GReeeeN的。同样，歌是好的，我是烂的，害
空飛ぶ電車《飞行列车》，遊助给JR東日本新幹線PV唱的曲子。有的地方太快了，但是是一首我想唱的好歌。
一笑懸命《一笑悬命》，也是遊助的，可以说基本是最出名的曲子了吧？しょう既可以写成“生”，也可以写成“笑”，所以いっしょうけんめい既可以写成一生懸命，也可以写成一笑懸命。歌是好的，但是emmmm感觉唱不出来这么欢乐的效果。
7月29日《7月29日》，飯田里穂的歌，最早在Google Music里随机到的，感觉好听就存着了。不太好唱的歌（指自己的声线）。
みちて、ひいて《看见、被看见》種崎敦美的歌，《大图书馆的牧羊人》小太刀的曲。怎么说呢，小太刀，我的小太刀...
逢うたび好きになって《每次相见就更加喜欢上你》，Bright的歌，我超，这个我超想唱的！但是，这个我真的唱不了，乌乌
恋ひ恋ふ縁《恋恋绿》（雾），千恋万花OP。是那种听到一次就想唱一次的曲子。但这个真的难唱，不太能想象得出来男人的声线唱这个啥样。
1年2ヶ月20日《1年2个月20天》，也是Bright的。咋说呢。这个也不是唱不了，不过先扔在这吧。
玻璃の空《玻璃之空》，小清水亚美，这名字好老了，这歌也好老了。08年的歌，现在听还是很有感觉。可惜声线不合适，要不然这首歌我也很想唱唱看。

Para Para Sakura《Para Para Sakura》，郭富城的歌。老实说郭富城是不是也是国语歌手里比较流弊的？这歌也好老了，或者说比起唱歌更想跳舞吧。乖乖龙地洞这个好像也是许多年前的广场舞歌曲。放第二个区域吧www
加賀岬《加贺岬》，舰C加贺的角色曲。最近一段时间就总在哼哼这个啊。感觉不太会唱，把握不太好怎么唱。含义也不是那么合适的歌。只是感觉最近总在哼哼。
